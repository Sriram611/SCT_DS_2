# Titanic EDA Project

This project demonstrates **Data Cleaning and Exploratory Data Analysis (EDA)** on a sample Titanic dataset using **Python (Anaconda / Jupyter Notebook)**.

## Project Files
- titanic_eda.ipynb
- titanic_sample_dataset.csv
- README.md

## How to Run
1. Open Anaconda Navigator
2. Launch Jupyter Notebook
3. Open titanic_eda.ipynb
4. Run all cells

## Description
This project cleans missing values and explores relationships between features such as survival, gender, class, age, and fare.
